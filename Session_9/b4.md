# Hệ thống Quản lý Khóa học Online

| **Entity (Thực thể)** | **Attribute (Thuộc tính)** | **Kiểu dữ liệu**                       | **Vai trò / Ý nghĩa**                                   |
| --------------------- | -------------------------- | -------------------------------------- | ------------------------------------------------------- |
| **User**              | user_id                    | INT                                    | Khóa chính, định danh duy nhất cho mỗi người dùng       |
|                       | full_name                  | VARCHAR(100)                           | Tên đầy đủ của người dùng                               |
|                       | email                      | VARCHAR(100)                           | Địa chỉ email dùng để đăng nhập và nhận thông báo       |
|                       | role                       | ENUM('student', 'instructor', 'admin') | Xác định vai trò của người dùng trong hệ thống          |
| **Course**            | course_id                  | INT                                    | Khóa chính, định danh duy nhất cho mỗi khóa học         |
|                       | course_name                | VARCHAR(150)                           | Tên của khóa học                                        |
|                       | description                | TEXT                                   | Mô tả nội dung, mục tiêu khóa học                       |
|                       | instructor_id              | INT (FK → User.user_id)                | Xác định giảng viên phụ trách khóa học                  |
| **Enrollment**        | enrollment_id              | INT                                    | Khóa chính, định danh duy nhất cho mỗi lượt đăng ký học |
|                       | user_id                    | INT (FK → User.user_id)                | Người dùng tham gia khóa học                            |
|                       | course_id                  | INT (FK → Course.course_id)            | Khóa học mà người dùng đăng ký                          |
|                       | enrollment_date            | DATE                                   | Ngày người dùng đăng ký khóa học                        |
